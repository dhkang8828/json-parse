# 드론 + RL 아이디어 분석

---

## 기술적으로 가능한가?

**결론: 충분히 가능합니다. 이미 연구/상용화 진행 중**

### 현재 드론에 RL이 쓰이는 곳

| 분야 | 내용 | 주체 |
|------|------|------|
| 자율 비행 경로 최적화 | 장애물 회피, 최단경로 탐색 | 대학 연구실, DJI |
| 드론 레이싱 | RL 에이전트가 인간 파일럿을 이김 | UZH(스위스), 2023년 Nature 게재 |
| 군집 비행 | 여러 드론이 협력해서 대형 유지 | DARPA, MIT |
| 배송 드론 | 착륙 지점 자율 선택 | Amazon Prime Air |
| 드론 전투 | 공중 도그파이트 RL | DARPA AlphaDogfight |

> 2023년 스위스 UZH 연구팀이 **RL 드론이 세계 챔피언 레이싱 파일럿을 이긴 것**이 Nature에 실림

## 부업으로 연결 가능한 방향

### A. 드론 레이싱 RL 에이전트 개발 & 시연 콘텐츠 ★★★

```
시뮬레이터(AirSim, Gazebo)에서 RL로 드론 학습
→ 유튜브 시연 영상 (조회수 잘 나오는 주제)
→ 깃허브 오픈소스 → 스타 → 강의로 연결
→ "내가 만든 AI 드론이 나를 이겼다" 콘텐츠
```

### B. 드론 + RL 교육 키트 ★★★

```
저가 드론(Tello, ~10만원) + Raspberry Pi
→ RL로 자율비행 가르치는 교육 패키지
→ 대학교 실습 교재 or 학원 교육 과정
→ 하드웨어 + 소프트웨어 묶어서 판매
```

### C. 시뮬레이터 기반 SaaS ★★★

```
브라우저에서 드론 RL 학습 환경 제공
→ 코드 짜서 내 드론 에이전트 훈련
→ 리더보드로 경쟁 유발
→ IDEA.md 변형 아이디어 3 (전략 코딩 대결)의 드론 버전
```

## 현실적인 장벽

### 하드웨어 진입 장벽

```
시뮬레이터   → 무료, 당장 시작 가능 (AirSim, Gazebo, PyBullet)
실제 드론    → 추락/파손 위험, 개발 비용 증가
실외 비행    → 항공법 규제 (도심 비행 허가 필요)
```

### Sim-to-Real Gap

```
시뮬에서 완벽히 날아도 실제 드론에선 바람/진동/센서 노이즈로 실패
이걸 극복하는 것 자체가 연구 주제
```

---

## 본인 배경과의 연결고리

```
C/C++ + 리눅스
    ↓
드론 펌웨어는 C/C++ 기반 (PX4, ArduPilot)
실시간 제어 시스템 경험이 그대로 활용됨

DRAM 테스트 (실시간 시스템)
    ↓
드론 제어도 실시간 제어 — 사고방식이 동일
```

> 이 분야에서 본인은 일반 ML 개발자보다 유리한 위치입니다.

---

## 추천 시작점

```
1단계  AirSim or Gazebo 시뮬레이터 세팅 (무료, 1주일)
2단계  OpenAI Gym 드론 환경에서 PPO로 호버링 학습 (2~3주)
3단계  유튜브/블로그에 과정 기록 (콘텐츠 자산화)
4단계  Tello 드론 구매 후 Sim-to-Real 도전 (10만원)
5단계  교육 키트 or 강의로 수익화
```
