# 사람들이 불편해하고 하기 싫은 일들

> 창업의 본질 = 사람들이 불편해하고 하기 싫은 일을 대신 해주는 것

---

## 1. 귀찮고 시간 낭비인 일

| 불편함 | 현재 해결책 | 기회 |
|--------|------------|------|
| 관공서 서류 발급/제출 | 직접 방문 | 대행 서비스 |
| 세금 신고 (프리랜서, 사업자) | 세무사 비쌈 | 저가 자동화 툴 |
| 보험 비교/정리 | 직접 비교 너무 복잡 | 자동 분석 서비스 |
| 이사 견적 비교 | 여러 곳 전화 | 자동 비교 플랫폼 |
| 중고거래 가격 조사 | 직접 검색 | 시세 자동 분석 |

---

## 2. 배우기 어렵고 진입장벽 높은 일

| 불편함 | 현재 해결책 | 기회 |
|--------|------------|------|
| 계약서 검토 | 변호사 비쌈 | AI 계약서 분석 |
| 주식/부동산 분석 | 공부해야 함 | 자동 리포트 |
| 영상 편집 | 툴 배우기 어려움 | AI 자동 편집 |
| 엑셀/데이터 정리 | 시간 오래 걸림 | 자동화 툴 |
| 영어 이메일 작성 | 어색하고 불안함 | 비즈니스 영어 교정 |

---

## 3. 심리적으로 하기 싫은 일

| 불편함 | 현재 해결책 | 기회 |
|--------|------------|------|
| 나쁜 소식 전달 (거절 문자 등) | 직접 보냄, 스트레스 | 대신 써주는 서비스 |
| 중고 물건 흥정/협상 | 직접 해야 함 | 협상 대행 |
| 악성 댓글/리뷰 대응 | 직접 써야 함 | AI 대응 초안 작성 |
| 구독 서비스 해지 | 전화 연결 귀찮음 | 해지 대행 |
| 헬스장 PT 식단 관리 | 매번 기록 귀찮음 | 자동 트래킹 |

---

| 특허 출원 초안 | 변리사 수백만원 | AI 초안 작성 |
| 인테리어 견적 비교 | 여러 업체 연락 | 자동 매칭 |
| 법률 서류 초안 | 변호사 비쌈 | AI 초안 서비스 |

## 6. 육체적으로 하기 싫은 일 (O2O 기회)

## 본인 스택으로 바로 만들 수 있는 것

```
개발자 + RL 관점에서 가장 현실적인 것

★★★★★  SNS 콘텐츠 자동 생성/스케줄링
          → AI로 글 생성 + RL로 최적 게시 시간 학습
          → 소상공인 타겟, 월 구독 모델

★★★★   엑셀/데이터 정리 자동화
          → 비개발자들이 가장 많이 겪는 고통
          → 파일 업로드 → 자동 정리 → 다운로드

★★★★   쇼핑몰 상품 설명 자동 생성
          → 사진 업로드 → AI가 상세페이지 초안 작성
          → 스마트스토어 셀러 타겟, 수요 매우 큼

★★★    회의록 자동 요약
          → 녹음 업로드 → STT → AI 요약
          → 기업 B2B 타겟
```
