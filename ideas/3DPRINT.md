# 3D 프린팅 주문 제작 웹앱 아이디어 분석

## 기본 구조

```
고객이 웹앱에서 도면 업로드 (STL, OBJ 등)
    → 견적 자동 산출 (재료비 + 시간 + 배송비)
    → 결제
    → 집에서 3D 프린팅
    → 포장 후 배송
```

---

## 긍정적인 면

### 1. 진입 장벽이 낮음
- 3D 프린터 중급 기종 50~150만원 (Bambu Lab X1C 등)
- 소재비 (PLA 1kg = 2~3만원) → 마진 충분
- 웹앱은 본인이 직접 개발 가능 → 플랫폼 비용 0원

### 2. 수요가 실제로 있음
- 소량 제작이 필요한 개인/소기업 (시제품, 부품 교체, 피규어)
- 기존 업체들은 MOQ(최소 주문 수량) 있거나 단가가 높음
- 개인 맞춤 제작 수요 꾸준히 증가 중

### 3. 확장 가능성
```
초기  → 1대로 시작
성장  → 프린터 추가 도입
확장  → 외주 프린터 네트워크 연결 (마켓플레이스화)
```

---

## 현실적인 문제점

### 1. 단가 경쟁이 심함
| 경쟁자 | 현실 |
|--------|------|
| 중국 제조 (JLCPCB, PCBWay 등) | 한국보다 훨씬 저렴한 단가 |
| 국내 3D 프린팅 업체 | 이미 다수 존재 (3DINN, 3D프린팅코리아 등) |
| 개인 크리에이터 (크몽, 숨고) | 재능마켓에서 이미 서비스 중 |

### 2. 도면 검수 문제
```
고객 도면이 잘못됐을 때 → 누구 책임?
출력 실패 시 → 재출력 비용은?
저작권 침해 도면 업로드 시 → 법적 리스크
```

### 3. 운영 현실
```
프린팅 시간  → PLA 기준 소형 10cm 물체 = 2~4시간
             → 동시에 여러 주문 들어오면 병목
배송         → 포장/택배 발송이 생각보다 번거로움
불량률       → 출력 실패 시 재료비 + 시간 손실
```

### 4. 수익성 계산

```
소형 출력물 기준 (10cm 이하)
  재료비       약 500~1,000원
  전기세       약 200원
  포장/택배    약 3,500원
  시간 (2시간) 본인 인건비
  ────────────────────────
  최소 판매가  8,000~15,000원은 받아야 남음

중형 출력물 (20cm 이상)
  재료비       약 3,000~8,000원
  전기세       약 500원
  포장/택배    약 5,000원
  ────────────────────────
  최소 판매가  20,000~40,000원
```

> 건당 마진은 괜찮지만 **볼륨(주문량)이 핵심** — 월 50건 이상은 돼야 의미 있는 수익

## 차별화 전략

### 1. 틈새 시장 집중
```
❌ 뭐든 다 찍어줌 (가격 경쟁에서 짐)
✅ 특정 카테고리 전문화

예시:
  - RC카/드론 부품 전문
  - 미니어처/피규어 전문
  - 산업용 지그/픽스처 전문  ← 본인 반도체 배경 활용 가능
  - 반려동물 용품 커스텀
```

### 2. RL을 붙이면 차별화 가능 ★
```
도면 업로드 시 RL/ML로 자동 슬라이싱 최적화
  → 출력 시간 단축
  → 재료 사용량 최소화
  → 견적 자동 산출 정확도 향상

"AI가 최적화한 3D 프린팅 서비스" → 마케팅 포인트
```

### 3. 웹앱 퀄리티로 승부
```
경쟁사 대부분 → 이메일로 도면 받고 수동 견적
우리 서비스   → 업로드 즉시 자동 견적 + 3D 미리보기
              → 고객 경험 차원이 다름
```

## 추천 MVP

```
1단계  3D 프린터 1대 구매 (Bambu Lab A1 Mini, 50만원대)
2단계  웹앱 제작
       - STL 파일 업로드
       - 부피 기반 자동 견적 계산
       - 결제 (토스페이먼츠)
       - 주문 현황 트래킹
3단계  크몽/숨고에서 먼저 수요 테스트 (웹앱 없이)
4단계  수요 확인되면 웹앱 정식 오픈
```

> 3단계를 먼저 하는 게 핵심 — 웹앱 만들기 전에 실제 수요가 있는지 검증

## 기술 스택

결제         토스페이먼츠 or 아임포트
